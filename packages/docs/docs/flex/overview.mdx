## Overview

The **Sortable Flex** component provides a flex container where items can be reordered through drag and drop interactions. It is more advanced than the Sortable Grid component and allows for more complex flexbox layouts.

:::important

**Sortable Flex** component layout calculation is much more complex than the **[Sortable Grid](/docs/grid/overview) component**. It is recommended to use the **[Sortable Grid](/docs/grid/overview)** component for most use cases.

:::
